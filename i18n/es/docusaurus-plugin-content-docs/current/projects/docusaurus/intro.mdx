¡Bienvenido a mi sitio web personal! Aquí he aprovechado el poder de [Docusaurus](https://docusaurus.io) para crear una plataforma completa, receptiva y fácil de usar. Aquí puedes explorar mi documentación, blog y otros contenidos, todo organizado y presentado con las potentes funciones de Docusaurus.

La nueva versión está aquí: [Docusaurus 3.0](https://docusaurus.io/docs/3.0.1).

## ✨ Características

- 🦖 **Docusaurus** - Basado en Docusaurus, proporcionando potentes funciones de generación de documentación y blogs.
- ✍️ **Markdown** - Escritura fácil con Markdown.
- 🎨 **Elegante** - Limpio, hermoso, prioriza la experiencia de lectura.
- 🖥️ **PWA** - Soporte para PWA, instalable, usable sin conexión.
- 🌐 **i18n** - Soporte para internacionalización.
- 💯 **SEO** - Optimización para motores de búsqueda, fácil de indexar.
- 📊 **Google Analytics** - Compatible con Google Analytics.
- 🔎 **Búsqueda de Texto Completo** - Compatible con [Algolia DocSearch](https://github.com/algolia/docsearch).
- 🚀 **Integración Continua** - Soporte para CI/CD, despliegue automático de actualizaciones.
- 🏞️ **Vista de Inicio** - Muestra las últimas entradas del blog, exhibiciones de proyectos, rasgos personales, stacks tecnológicos, etc.
- 🗃️ **Vistas de Entradas del Blog** - Diferentes vistas de las entradas del blog: lista, cuadrícula.
- 🌈 **Navegación de Recursos** - Recopila y comparte recursos útiles e interesantes.
- 📦 **Portafolio** - Muestra tus proyectos, se puede utilizar como un portafolio.
